Projects

Software

  • PSYCHIC, an extractive question-answering neuro-symbolic model that answers a knowledge graph question with a SPARQL query
  • FinISH, a model that classifies financial terms from domain ontology classes

  • FNP T5 D2T complete, a complete T5-base data to text model specialized for Finance NLG
  • FNP T5 D2T simple, a simple T5-base data to text model specialized for Finance NLG
  • Text smoothing, a generative model for financial text rewriting
  • Code of other experiments on GitHub
